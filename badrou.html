SYNTAXES DU LANGAGE HTML

1) Qu'est-ce que le Langage HTML?

         HTML signiﬁe HyperText Markup Language, ce qui se traduit par langage de balisage hypertexte en français. C’est le langage standard pour créer des pages web. HTML est utilisé pour baliser un contenu, c’est à dire pour le structurer et lui donner du sens. Il déﬁnit la structure et le contenu d'une page web, en indiquant au navigateur web comment aﬃcher le texte, les images, les vidéos et autres éléments.
Le langage HTML est utilisé pour structurer et donner du sens au contenu d'une page web, en le balisant. Il sert à indiquer aux navigateurs quel texte doit être considéré comme un paragraphe, quel texte doit être considéré comme un titre, que tel contenu est une image ou une vidéo.

2)Fonctionnement du langage HTML:éléments , balises, attributs.

-Les elements

        Pour déﬁnir la structure et le contenu d'une page web, on écrit chaque élément de la page web avec le langage HTML dans un document HTML (nom_du_document.html). Ainsi un texte sous forme de paragraphe sur une page web serait représenté par un élément paragraphe (élément p) dans le document HTML. De même une image, un titre, une liste d’une page web seraient représentés par l’ élément correspond dans un document HTML.

-Les balises

        Un élément en HTML est structuré par des balises. Les balises décrivent le type de contenu et sa présentation.Une balise en HTML représente un élément et est délimité par des chevrons <>. Par exemple, la balise <p> indique un paragraphe, tandis que la balise <h1> indique un titre principal.<br>Généralement, un élément HTML est constitué d'une paire de balises: une balise ouvrante (par ex: <p>) et une balise fermante (</p>). La balise fermante porte le nom de la balise ouvrante correspondant mais a en plus un caractère Slash “/” au début.
Certains éléments ne nécessitent qu'une seule balise, appelée balise auto-fermante ou balise solitaire. Pour ces éléments, il est conseillé d'ajouter un slash (/) à la ﬁn de la balise.

Exemples d'éléments nécessitant une seule balise :

<br> : pour insérer un saut de ligne
<hr> : pour insérer un trait horizontal
<img> : pour insérer une image
<input> : pour créer un champ de saisie

3)Structure de base d’une page HTML

        Le langage HTML a l’avantage d’être simple à utiliser. En eﬀet, tous les documents HTML suivent une structure de base qui est toujours le même pour tous. Il s’agit d’un <squelette> composé d’éléments déterminé sur lequel est formé tout document HTML.

-Déclaration du type de document (Doctype)

        Tout document HTML commence par la déclaration de type de document, qui indique au navigateur web la version de HTML utilisée. La syntaxe standard est la suivante :<br> HTML<br> <!DOCTYPE html>

-Balise <HTML>

        La balise <html> englobe l'ensemble du contenu HTML d'une page web. Elle sert de conteneur principal pour tous les autres éléments HTML.
<html>
...
</html>

-Balise <head>

        La balise <head> contient des informations métadonnées sur la page web, telles que le titre, l'encodage de caractères et les liens vers des feuilles de style CSS externes.

-Balise <body>

        La balise <body> contient le contenu visible de la page web, c'est-à-dire le texte, les images, les liens et autres éléments que les utilisateurs voient et interagissent avec.

3)Les liens hypertextes

Les liens hypertextes, ou hyperliens, sont des éléments essentiels du web qui permettent aux utilisateurs de naviguer entre diﬀérentes pages ou sections d'une même page. En HTML, ils sont créés à l'aide de l'élément <a>.

Exemple:<a href="https://www.example.com">Lien vers un site externe</a>

        Un lien hypertexte HTML se compose de deux parties principales :
<a href="https://www.example.com">Lien vers un site externe</a>
❏ Le texte ou l'image qui sera aﬃché et cliquable pour l'utilisateur. C'est placé entre les balises <a>....</a>. ❏ L'attribut href: indique l'URL de la page web vers laquelle le lien pointe. Il est obligatoire pour que le lien fonctionne.
Parmi les types de liens hypertextes en HTML, on peut distinguer d’abord :
❏ Les liens internes: Ils pointent vers une autre section de la même page web.
❏ Les liens externes: Ils pointent vers une page web sur un autre domaine.

-Les liens internes

Les liens internes, ou liens intra-page, permettent de naviguer entre diﬀérentes sections d'une même page web.

-Les liens externes

Les liens externes, ou liens hypertextes sortants, permettent de pointer vers des pages web situées sur un domaine diﬀérent de celui de la page actuelle.

-Les liens de type ancre

Les ancres nommées, ou ancres HTML ou repères, servent à créer des points d'ancrage dans une page web.

Les liens de type ancre: fonctionnement
1. Déﬁnir une ancre nommée: L'ancre est déﬁnie en ajoutant l'attribut id à un élément HTML.
<h2 id="titre-section">Titre de la section
2. Créer un lien interne: Le lien interne est créé à l'aide de l'élément et de l'attribut href. <a href="#titre-section">Aller à la section "Titre de la section" Lorsque l'utilisateur clique sur le lien interne, le navigateur web fait déﬁler la page vers l'élément HTML qui possède l'ID correspondant au nom de l'ancre (titre-section dans cet exemple).

4)Attributs des liens hypertextes

Les liens hypertextes, qu'ils soient internes, externes ou utilisés avec des ancres nommées, partagent un attribut commun essentiel : href. Cependant, des attributs supplémentaires facultatifs peuvent être utilisés pour aﬃner leur comportement et enrichir leurs fonctionnalités.

-Attributs des liens hypertextes: href

L’attribut href la seule obligatoire pour tous les liens spéciﬁe la destination du lien, qu'il s'agisse d'une page interne (commençant par #) ou d'une page externe (URL complète).
Syntaxe: href="valeur_cible"

Exemple:



-Attributs de liens hypertextes: target

L’attribut facultatif target permet de contrôler dans quelle fenêtre ou onglet du navigateur le lien s'ouvrira.
Syntaxe: <a href="cible" target="_blank">Lien dans un nouvel onglet

Valeurs possibles:

➲ _self (par défaut): Ouvre le lien dans l'onglet actuel.
➲ _blank: Ouvre le lien dans un nouvel onglet.
➲ _parent: Ouvre le lien dans la fenêtre parente.
➲ _top: Ouvre dans la plus haute fenêtre du navigateur.

-Attributs de liens hypertextes: rel

Fournit des informations sur la relation entre la page actuelle et la page de destination:
Syntaxes: <a href="cible" rel="noopener noreferrer">Lien sans ouverture et sans suivi

-Attributs de liens hypertextes: title

Ajoute un texte informatif qui s'aﬃche comme info-bulle lorsque l'utilisateur survole le lien.
Syntaxe: <a href="cible" title="Description de la page cible">Lien avec info-bulle

-Attributs de liens hypertextes: rel

Valeurs possibles:

➷ noopener: Empêche l'ouverture de la page cible dans une nouvelle fenêtre.
➷ noreferrer: Indique que la page cible ne doit pas être associée à la page actuelle pour le suivi des références.

-Attributs de liens hypertextes: download

Indique que le lien doit être téléchargé plutôt qu'ouvert dans le navigateur: Utile pour les ﬁchiers téléchargeables (PDF, images, etc.).

Syntaxe: <a href="ﬁchier.pdf" download="nom-ﬁchier.pdf">Télécharger un ﬁchier

LES ELEMENTS DU DU LANGAGE HTML (image, span, div, header, section, footer)

L'élément <img>

L'élément <img> est l’élément HTML utilisé pour intégrer des images dans des pages web. Il permet d'aﬃcher des images provenant de divers formats, tels que JPEG, PNG, GIF, SVG, etc.

Syntaxe de base:

<img src="URL_image" alt="Texte alternatif">

*Attributs de l’élément img: src

Lorsque le navigateur rencontre un élément <img> dans le code HTML, il commence par analyser ses attributs.
L'attribut src est obligatoire et indique l'URL de l'image à aﬃcher.

Syntaxe: <img src="URL_image">

*Attributs de l’élément img: alt

L’Attribut alt permet d’indiquer un texte alternatif qui décrit l'image et s'aﬃche si l'image ne peut pas être chargée ou pour les utilisateurs d'outils d'assistance. Cet attribut est fortement recommandé pour l'accessibilité.

Syntaxe: <img src="URL_image" alt="Texte alternatif">

*Attributs de l’élément img: title

L’Attribut title permet d’indiquer un texte informatif qui apparaît comme info-bulle lorsque la souris survole l'image.

Syntaxe: <img src="URL_image" alt="Texte alternatif" title=”Titre informatif”>

Exemple d'utilisation de l’élément img

Galerie d'images
<div class="galerie">
<img src="image1.jpg" alt="Image 1">
<img src="image2.jpg" alt="Image 2">
<img src="image3.jpg" alt="Image 3">

-L’élément <span>

L'élément <span> est un élément générique en ligne (inline) utilisé pour marquer une partie de texte ou un élément HTML pour le styliser ou lui ajouter des attributs sans créer de nouvelle ligne de contenu.

Syntaxe de base:

<span>Texte à styliser</span>

Exemple d'utilisation de l'élément span

Il est souvent utilisé pour appliquer des styles spéciﬁques à un mot, une phrase ou un groupe de caractères précis au sein d'un paragraphe ou d'un autre élément de texte.

Exemples d'utilisation:
colorier un texte en rouge:
<span style="color: red;">Attention, cette zone est dangereuse.</span>

-L’élément div

L'élément <div> (division) est un élément générique de type bloc (block-level) utilisé pour regrouper et organiser des sections de contenu dans une page HTML.

Syntaxe de base:

<div>contenue a organiser</div>

Exemple d'utilisation de l'élément div

Il sert de conteneur pour d'autres éléments HTML et permet de structurer le contenu de manière logique et visuelle. .

Exemples d'utilisation:
Grouper des éléments pour le style:

<h2>Produit du mois

<header>contenue de l'en tete</header>

-L'élément <Section>

L'élément <section> représente une section générique de contenu dans un document HTML. Il sert à regrouper et organiser des sections thématiques ou logiques d'une page.

Syntaxe de base:

<section>

</section>

Exemple d'utilisation de l'élément Section

-L'élément <Footer>

L'élément <footer> représente le pied de page d'une section ou de la page entière dans un document HTML. Il sert à regrouper des informations de clôture, telles que les droits d'auteur, les liens vers les pages connexes, les coordonnées de contact ou d'autres éléments.

Syntaxe de base:

<Footer>
Contenu du pied de page
</Footer>

Exemple d'utilisation de l'élément Footer

Pied de page de page web avec droits d'auteur et liens:

<li><a href="terms.html">Conditions générales</a></li>
</ul>
</footer>
